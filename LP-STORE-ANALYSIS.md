## 三、API 端点详解

### 3.1 核心 API 列表

| 端点 | 方法 | 功能描述 |
|-----|------|---------|
| `/api/lp/faction` | GET | 获取所有势力和公司列表 |
| `/api/lp/offer` | GET | 获取 LP 商店物品及计算数据 |
| `/api/common/region` | GET | 获取区域信息 |
| `/api/images/{type}/{id}` | GET | 获取图标资源 |

### 3.2 LP Offer API 参数

```
GET /api/lp/offer?
  regionId=10000002      # 区域 ID
  scope=0.05            # 价格范围（±5%）
  corporationId=1000002 # 公司 ID
  lang=zh               # 语言（中文）
  days=7                # 数据周期（7天）
  tax=0                 # 税费（0%）
  productPrice=buy      # 产品价格参考：买入价
  materialPrice=sell    # 材料价格参考：卖出价
```

## 四、数据表格结构

### 4.1 列定义（13列）

| 列名 | 数据类型 | 含义 | 计算方式 |
|-----|--------|------|---------|
| 物品 | String | 商品名称 | ESI 数据库 |
| 数量 | Integer | 单次兑换数量 | LP 商店规则 |
| **LP成本** | Integer | 忠诚点成本 | 直接显示 |
| **ISK成本** | Integer | ISK 游戏币成本 | 直接显示 |
| **材料成本** | Integer | 需要的材料总值 | 材料价格 × 数量 |
| **单价** | Integer | 产品市场单价 | 市场历史价格均值（7天） |
| **收入** | Integer | 单价 × 数量 | 计算字段 |
| **利润** | Integer | 收入 - ISK成本 - 材料成本 | **重要指标** |
| 交易量 | Integer | 7天交易量（单位） | 市场统计 |
| 出货指数 | Integer | 交易额排名指数 | 市场热度 |
| **LP比例** | Integer | 利润 / LP成本 | **ROI 指标** |
| 物品搜索 | String | 搜索框 | 客户端过滤 |

### 4.2 排序规则

表格默认按照 **LP比例（利润/LP成本）** 降序排列，帮助用户快速找到最高回报的兑换项。

## 五、核心计算逻辑

### 5.1 利润计算公式

```javascript
// 基础计算
收入 = 单价 × 数量
材料成本 = SUM(材料价格 × 数量)
利润 = 收入 - ISK成本 - 材料成本
LP比例 = 利润 / LP成本  // 每点 LP 的 ISK 价值
```

### 5.2 计算示例

**示例 1：物流关系学（无材料成本）**
```
单价: 57,000,000 ISK
数量: 1
ISK成本: 10,000,000
材料成本: 0
收入: 57,000,000
利润: 57,000,000 - 10,000,000 - 0 = 47,000,000 ISK
LP比例: 47,000,000 / 15,000 = 3,133 ISK/LP
```

**示例 2：加达里海军重型导弹发射器（含材料成本）**
```
数量: 1
ISK成本: 9,600,000
材料成本: 109,373,423
  - 联邦海军舰队舰长勋章I × 63 = 78,060,857
  - 联邦海军舰队少校勋章I × 106 = 31,120,566
  - 重型导弹发射器 I × 1 = 192,000
单价: 171,300,000
收入: 171,300,000
利润: 171,300,000 - 9,600,000 - 109,373,423 = 52,326,577 ISK
LP比例: 52,326,577 / 24,000 = 2,180 ISK/LP
```

## 七、用户交互流程

```
1. 选择势力 → 下拉菜单展开第一层
   ↓
2. 选择公司 → 下拉菜单展开第二层
   ↓
3. 触发 API 请求
   GET /api/lp/offer?corporationId={selectedCompanyId}&...
   ↓
4. 后端计算和数据聚合
   - 获取该公司的所有 LP offer
   - 查询市场价格（7天内）
   - 计算利润和 ROI 指标
   ↓
5. 渲染数据表格
   - Vue 组件更新响应式数据
   - 分页显示（默认25条/页）
   - 动态渲染行数据
   ↓
6. 用户可选操作
   - 搜索物品名称（客户端过滤）
   - 点击列名排序
   - 查看下一页
   - 点击"订单"按钮查看市场订单
```

## 十、对本项目的参考价值

### 10.1 实现类似功能需要的数据

1. **LP 商店数据**
   - 物品 ID、名称
   - LP 成本
   - ISK 成本
   - 材料需求列表

2. **势力-公司关系**
   - 势力 ID、名称
   - 公司 ID、名称、所属势力

3. **市场价格数据**
   - 通过 EVE ESI API 或 ceve-market API 获取
   - 买入价/卖出价
   - 历史交易量

### 10.2 后端计算需求

- 定时抓取市场价格
- 计算材料成本、利润、LP 比例
- 缓存计算结果（Redis）

### 10.3 前端展示需求

- 级联选择器（势力 → 公司）
- 数据表格 + 排序/搜索
- 分页控制
